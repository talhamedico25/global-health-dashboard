# ðŸŒ Global Health Data Dashboard

An interactive dashboard that aggregates global health indicators from the World Bank API. Compare countries, track trends over time, and discover insights about health disparities worldwide.

## Features

- **9 Health Indicators**: Life expectancy, mortality rates, immunization coverage, GDP per capita, health expenditure, and more
- **Interactive Charts**: Line charts, bar charts, world maps, and correlation scatter plots
- **Filters**: Filter by region, income group, and time period
- **Country Comparison**: Compare up to 10 countries side-by-side
- **Data Export**: Download data as CSV for further analysis

## Quick Start

```bash
cd global-health-dashboard

# Activate virtual environment
source venv/bin/activate

# Run the dashboard
streamlit run app.py
```

Then open http://localhost:8501 in your browser.

## Available Indicators

- Life Expectancy at Birth
- Under-5 Mortality Rate
- Infant Mortality Rate
- DPT Immunization Coverage
- Measles Immunization Coverage
- GDP per Capita
- Health Expenditure (% of GDP)
- Physicians per 1,000 people
- Hospital Beds per 1,000 people

## Data Source

All data is fetched from the [World Bank Open Data API](https://data.worldbank.org/).

## Tech Stack

- **Streamlit** - Web UI framework
- **Pandas** - Data manipulation
- **Plotly** - Interactive visualizations
- **Requests** - API calls
- **Requests-Cache** - API response caching
